# IntelBot – Новостной аналитик в Telegram

Мультиарендный бот: собирает новости из Telegram-каналов, RSS и сайтов,
анализирует через RAG + LLM, отправляет саммари, тональность и теги.

**Бизнес-модель:** один код — отдельный конфиг на каждого клиента.
Ниши: строительство, FMCG, мониторинг конкурентов/тендеров/цен.

## Принципы разработки

- **asyncio** везде где есть I/O; блокирующие вызовы — через `asyncio.to_thread`
- **Type hints** обязательны для всех функций и методов
- **Loguru** для логирования; каждый модуль — свой логгер с именем `__name__`
- **Pydantic v2** для валидации конфигов клиентов при загрузке
- **PEP 8**, docstrings для всех публичных функций и классов
- Обработка ошибок: повторы с экспоненциальной задержкой при парсинге, fallback при LLM
- **Мультиарендность:** все операции с БД и ChromaDB фильтруются по `client_id`
- Токены и секреты — только в `.env`, не в репозитории
